## Optical Properties

### Luster Types

| Type | Description | Examples |
|------|-------------|----------|
| Metallic | Reflective like metal | Gold, Silver, Copper |
| Vitreous | Glass-like shine | Quartz, Feldspar |
| Adamantine | Diamond-like brilliance | Diamond, Zircon |
| Pearly | Pearl-like sheen | Talc, Gypsum |
| Resinous | Resin-like appearance | Amber, Sulfur |

### Optical Phenomena

- **Pleochroism**: Different colors when viewed from different angles
- **Chatoyancy**: Cat's eye effect (parallel inclusions)
- **Asterism**: Star effect (intersecting inclusions)
- **Labradorescence**: Iridescent play of colors
- **Fluorescence**: Emission of visible light under UV
